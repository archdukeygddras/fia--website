# 🎓 Fulfilment International Academy

A premier institution for higher education, now featuring a fully digital student management system.

[![Live Site](https://img.shields.io/badge/Live-Demo-gold?style=for-the-badge&logo=github)](https://archdukeygddras.github.io/fia--website/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg?style=for-the-badge)](LICENSE)

Fulfilment International Academy provides world-class education, cutting-edge programs, and a modern student record system powered by a cloud-based backend.

## 🚀 Key Features

-   **📋 Student Registry:** Comprehensive database of enrolled students.
-   **✏️ Quick Registration:** Smart form for enrolling students in seconds.
-   **⚡ Real-Time Updates:** Instant data synchronization powered by Supabase.
-   **🔐 Secure Staff Login:** Encrypted access for authorized academy staff.
-   **🔍 Smart Search & Filter:** Effortlessly navigate hundreds of records.
-   **📊 Cloud Powered:** Accessible from any device, anywhere in the world.

## 🛠️ Tech Stack

-   **Frontend:** HTML5, Vanilla CSS3 (Custom Design System)
-   **Logic:** Modern JavaScript (ES6+)
-   **Backend:** [Supabase](https://supabase.com/) (PostgreSQL + Auth)
-   **Animations:** Intersection Observer API & CSS Keyframes

## 📂 Project Structure

```text
├── css/
│   ├── style.css      # Core Design System & Components
│   └── landing.css    # Landing Page specific styles
├── js/
│   ├── app.js         # Supabase Client & Shared Utilities
│   ├── landing.js     # Intersection Observer logic
│   ├── login.js       # Authentication logic
│   ├── register.js    # Student enrollment logic
│   └── students.js    # Registry management & filtering
├── index.html         # Main Dashboard
├── landing.html       # Public Marketing Page
├── login.html         # Staff Portal Access
├── register.html      # Enrollment Form
└── students.html      # Student Management Interface
```

## 🛠️ Local Development

1. **Clone the repository:**
   ```bash
   git clone https://github.com/archdukeygddras/fia--website.git
   ```
2. **Open `landing.html`** in any modern web browser.
3. **No Build Step Required:** This project uses pure Vanilla JS and CSS.
